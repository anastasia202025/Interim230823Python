1. Программа считывает данные из файла csv и создает список экземпляров класса Note. Данный список используется программой для работы с записями (добавление, удаление, редактирование). Сразу после прочтения исходного файла создается резервный файл bak.

2. Схема работы с уже существующими записями: поиск по фрагменту информации из нужной записи (из любого поля, включая дату и время) ==> запрос id нужной записи ==> запрос действия (редактирование, удаление) ==> само действие.

3. Вывод на экран всего списка заметок и добавление новой - из главного меню.

Модули, использованные в программе:
    main_notes - стартовый
    dump - работа с файлами
    Note - класс записей
    menu - модуль меню с сопутствующими методами
